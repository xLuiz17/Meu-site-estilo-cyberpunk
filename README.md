# Meu-site-estilo-cyber
Mini-site estático com recomendações de animes, filmes, HQs, jogos e outros conteúdos no estilo cyberpunk. Desenvolvido com HTML5 e CSS3, o projeto apresenta um visual moderno e dark, com listas, imagens, tabela e links para aprofundar o conhecimento no tema. Ideal para estudos e demonstração de organização de páginas web.
